# RoadDamageDetect
도로 상태 감지기 이 프로젝트는 Flask를 기반으로 한 웹 애플리케이션으로, 사용자가 업로드한 도로 사진에서 파손된 부분을 감지합니다. 

이 애플리케이션은 YOLO 모델을 사용하여 이미지에서 손상된 도로 부분을 감지하고 결과를 웹 페이지에 시각적으로 표시합니다.
